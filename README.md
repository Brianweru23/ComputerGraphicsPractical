# Architectural Design for 14-Floor Commercial Building
**Student ID**: BIT/2018/79502  
**Course**: Computer Graphics CAT  

## Project Overview
This repository contains the architectural and engineering design for a 14-floor commercial building. The design includes:
- **Typical Floor Plan**: Detailed layout of one floor including offices, corridors, stairs, and elevators.
- **Electrical Wiring Layout**: Power distribution, lighting, and socket outlets.
- **Water Supply and Plumbing Plan**: Vertical risers and horizontal distribution.
- **Wi-Fi and Network Cabling Plan**: Data points and Wi-Fi access point locations.

## Files Included
1. `BIT_2018_79502_COMPUTERGRAPHICS_CAT.dxf`: Editable CAD file (compatible with AutoCAD/Archicad).
2. `BIT_2018_79502_COMPUTERGRAPHICS_CAT.pdf`: Professional PDF export of the drawings.

## Design Standards
- **Layers**: Organized by discipline (A-WALL, E-POWR, P-WATR, T-DATA).
- **Scale**: 1:100 (Metric).
- **Symbols**: Standard architectural and engineering symbols used throughout.
